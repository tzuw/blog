Java 常用语法糖整理

January 11, 2020 约2分钟读完 大约493个字

释义

语法糖，指的是能让程序员更容易表达逻辑的语法，方便开发。

语法盐，指的是程序语言中的防呆设计。

语法糖精，程序语言的过度设计，让编程变得不直观。

举例 - 常用Java语法糖

  • switch支持 String 与枚举类型

  • 泛型

  • 自动装箱与拆箱：原始类型与对应的封装类互相转换。

  • 方法变长参数：允许函数传一个或者多个参数，且写法是一样的。

  • 枚举

  • 内部类

  • 条件编译断言

  • 数值字面量

  • for-each

  • try-with-resource

  • Lambda表达式
